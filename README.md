# 🛒 Digital Product Store

A modern, responsive digital product marketplace built with React and Tailwind CSS. Browse and purchase premium digital tools — from AI writing assistants to social media kits — all in one place.

---

## 🚀 Features

- 🛍️ **Add to Cart** — Seamlessly add any product to your cart and manage your selections in real time
- 💳 **Instant Checkout** — Proceed to checkout with a smooth modal confirmation experience
- 🔔 **Live Notifications** — Get instant toast alerts when adding or removing products from your cart

---

## ⚙️ Technologies Used

| Technology | Purpose |
|---|---|
| ⚛️ React.js | UI components & state management |
| 🎨 Tailwind CSS | Utility-first styling |
| 🌼 DaisyUI | Pre-built component library |
| ⚡ JavaScript ES6+ | Core logic & interactivity |
| 🔔 React-Toastify | Toast notification system |
| 📦 JSON | Local product data storage |
